## 写给准爸准妈

</br>
以上列表仅供参考，可以自行微调

希望各位准爸准妈宝宝都顺利出生，健康、可爱

如果有条件可以进产房陪产，准爸一定会有心灵震撼，准妈无数次的用尽全身力气，不抱怨不放弃，比男人都要坚韧，非常让人心疼和敬佩，请一定一定要爱这位为你生娃的妈妈，太不容易了

</br>

### 1.产后护理

如果有月嫂，可以浇月嫂进来帮忙，家人建议是比较懂一点的来

我这里是因为是疫情，医院陪护仅限1人，要熬夜辛苦几天

> 关于热水：不要直接用水龙头的水，而是热水放凉了，再兑点热的到合适温度即可

* 宝宝护理：比较单一，喂奶，第1天一般都没有奶，用奶粉看好比例，有的是30:1,；拍嗝，抱起来，轻轻拍后背，喝奶喝的猛了一般都有的；侧放着睡，保护头部；抱着的时候，一定要扶着脖子！他的脊椎还没有足够的力气；宝宝睡觉时，可以补一些宝宝的护理常识；每天可以去医院指定的地方给宝宝洗澡；手喜欢乱抓，用袜子套手上

* 宝妈护理：床上铺好护理垫，白天穿一次性内裤+卫生巾，晚上夜安裤，头两天恶露偏多的，及时更换，排干净宝妈身体才健康；不要一直问宝妈有没有奶水，容易造成心理压力，顺其自然即可，医生每天都会推来催乳仪器，也可喝些红糖水，其他的护理医生会说

顺产一般3天会办理出院手续，宝宝抽完足底血就可以出院了

</br>

### 2.出院后护理

</br>

和在医院差不多，多的是做饭和洗衣服，累一点

宝妈坐月子，别让她碰冷水和劳累，生产用力过度了，手脚都使不上力

宝爸要上班的话，可以让家人帮衬下

</br>

### 3.以后的事情

</br>

30天内记得宝宝的名字想好了，可以线上办理出生一件事，让他拥有姓名，我这里是浙里办就可以；其他的就按时打预防针即可，出院手册上有的
